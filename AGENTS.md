# Agents

This file provides an overview of custom agents for the Freiburg Run project.

## Overview

Custom agents are defined in `.github/agents/` as individual `.agent.md` files following VS Code best practices.

## Available Agents

- **Refactoring Agent**: Located at `.github/agents/refactoring-agent.agent.md`. Use for refactoring code without changing functionality.

## Usage

To use a custom agent, reference it by name in your queries or configurations, or use slash commands like `/refactoring-agent` in Copilot Chat.
